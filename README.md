# MythPoke Store Landing Page

Single-page landing site for MythPoke — premium extended Pokémon card frames.

## Getting Started

```bash
npm install
npm run dev
```

The dev script launches Next.js 15 with Turbopack (`next dev --turbo`). Visit http://localhost:3000 to view the landing page.

## Tech Stack

- Next.js 15 App Router
- React 18
- Tailwind CSS
- TypeScript

## Project Structure

- `app/` — App Router entry point, global styles, page composition.
- `app/components/` — Modular UI sections (Hero, Products, FAQs, Footer).
- `app/data/` — Sample catalog and FAQ content used by the landing page.
